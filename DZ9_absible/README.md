#### Ansible установка nginx
##### Как проверить работу 
	 1. Копирует Vagrantfile (стенд с 2-я хостами Centos). Запускаем 
	 2. Копируем каталог с Ролью nginx и run_rol_nginx.yml
	 3. Запускаем run_rol_nginx.yml
